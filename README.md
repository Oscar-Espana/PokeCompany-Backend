# PokeCompany - Backend

## Backend

Installation:

```bash
npm install
# or
yarn install
```

Run the backend:

```bash
npm run dev
# or
yarn dev
```

The backend is will running in the [http://localhost:3000](http://localhost:3000)

The sandbox is will running in [http://localhost:3000/graphql](http://localhost:3000/graphql)
